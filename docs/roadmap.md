# Roadmap

## Fase 0. Fundacion

Objetivo:

Definir con precision el lenguaje de negocio, actores, entidades y flujo central.

Entregables:

- modelo de dominio,
- mapa de estados,
- permisos por actor,
- wireframes base por rol.

## Fase 1. Operations Core

Objetivo:

Poder crear, asignar, ejecutar y revisar servicios de homecare.

Incluye:

- onboarding de actores,
- ordenes de servicio,
- matching basico,
- agenda,
- checklist,
- evidencia,
- incidencias,
- revision y aprobacion.

## Fase 2. Network Control

Objetivo:

Dar control real a la prestadora y visibilidad al centro.

Incluye:

- portal del centro,
- reemplazos y escalamiento,
- alertas de cobertura,
- score operativo,
- reportes por periodo.

## Fase 3. Solo Business Toolkit

Objetivo:

Convertir al cuidador en un operador mas confiable y mejor gestionado.

Incluye:

- gastos,
- kilometraje,
- perfil profesional,
- vencimientos,
- historial de ingresos y servicios,
- recordatorios administrativos.

## Fase 4. Financial Closure

Objetivo:

Reducir friccion en el cierre economico.

Incluye:

- consolidado de horas aprobadas,
- base para facturacion,
- base para liquidacion,
- manejo de excepciones economicas.

## Secuencia recomendada

No empieces por:

- historias clinicas,
- facturacion compleja,
- marketplace abierto,
- app para familias.

Empieza por:

- operaciones,
- trazabilidad,
- cumplimiento,
- cierre simple.
